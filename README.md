# Finca ERP – Currency Exchange Management System

## Overview
Finca ERP is a web-based application that simulates a currency exchange office. It allows users to perform currency conversions, store transactions, and view financial summaries.

## Features
- Currency exchange calculation
- Transaction history
- Dashboard analytics

## Technologies
Frontend:
- React
- Vite
- Axios

Backend:
- Node.js

Database:
- PostgreSQL

## How to Run

### Frontend
cd frontend
npm install
npm run dev

### Backend
cd backend
npm install
node server.js
